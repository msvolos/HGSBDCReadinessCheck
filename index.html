Admin Configuration

Assessment Settings

Edit question text, answer options, and scores (1–10) for each answer. Changes are saved automatically.

Set the Effort Score (1=minimal, 10=very high) and Impact Score (1=low, 10=transformational) for each recommended action. These control bubble placement in the Value vs. Effort matrix.

📊 Google Sheets Integration

Paste your Google Apps Script Web App URL to automatically log all assessment submissions to a Google Sheet.

Deploy an Apps Script as a Web App with doPost(e) to receive assessment data as JSON. The payload includes all scores, dimension results, and the full HTML report.

📁 Google Drive PDF Storage

When a webhook URL is configured, the PDF report HTML is also sent to your Apps Script, which can save it to a Google Drive folder.

💡 Your Apps Script should handle both Google Sheets logging (via SpreadsheetApp) and Drive storage (via DriveApp). The payload's report_html field contains the full printable PDF report.

⚠️

Reset All Settings to Defaults

This will restore all questions, answer options, scores, and action effort/impact values to their original defaults.

Export / Import Settings

Download your current settings as a JSON file for backup or sharing across deployments.
